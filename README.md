# TIL

### 📂 폴더 구조
```bash
└─ 📁 이니셜
    └─ 📄 년-월-일.md
```

### ⭐ conventions
- 처음 작성일때 : [이니셜] 날짜_생성
- 내용 추가일 때 : [이니셜] 날짜_수정

### 📜 templates
서식은 수정가능하나 아래 내용은 필수로 추가되어야 합니다.

```md
# ☀️ 오늘 한 일

# 🚩 내일 할 일
```
